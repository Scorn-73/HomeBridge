# HomeBridge
Smart home set up
- Notes/info regarding setting up my HomeBridge smart home


## Plugins

### Meross
- https://github.com/homebridge-plugins/homebridge-meross
  - Smart power strip
  - Smart Plug
  - Temp and humidity sensor

### Homebridge Dummy
- https://github.com/mpatfield/homebridge-dummy#readme
  - Dummy switches for advanced automations
    - Example Automations:
      Uses advanced timers, switches, and triggers to keep a vivarium at optimal conditions for a pet tortoise,
      creating alerts if temperature or humidity fall outside the preferred range.

### webOS TV
- https://github.com/merdok/homebridge-webos-tv#readme
  - Connect TV for advacned settings control

### Homebridge TAPO Camera
- https://github.com/kopiro/homebridge-tapo-camera#readme
  - Home Security Cameras

 ### Homebridge Dreo
 - https://github.com/zyonse/homebridge-dreo
   - Fan and Heater intergrations

### Homebridge Eufy Security
- https://github.com/homebridge-plugins/homebridge-eufy-security/wiki#
  - Video doorbell interbration with HomeKit Secure Video

 ### Homebridge Tuya
 - https://github.com/iRayanKhan/homebridge-tuya#readme
   - Smart light and Heater intergration

### Homebridge Smartthings oAuth Plugin
- https://github.com/aziz66/homebridge-smartthings#readme
  - Intergatrion for Smart lamps
    - oAuth 2.0 app creation required
